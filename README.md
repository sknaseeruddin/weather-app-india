# Weather App India

A responsive weather application built using HTML, CSS, and JavaScript that supports village, city, district, and current location search across India.

## Features
- Search weather by village, city, or district
- Current location weather
- India-focused geocoding support
- Map-based fallback search
- Temperature, humidity, wind speed, pressure
- Sunrise, sunset, visibility, coordinates
- Recent searches with localStorage
- Dynamic background based on weather
- Responsive design for desktop and mobile

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
- OpenStreetMap Nominatim API
